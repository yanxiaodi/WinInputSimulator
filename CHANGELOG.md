## [1.0.0] - 2025-01-XX

### Added
- Initial release of WinInputSimulator
- Keyboard simulation with Unicode support
- Mouse operations (click, move, scroll)
- Window management features
- Support for .NET 8.0, 9.0, and 10.0
- Virtual key code constants for easy access
- Key combination support (Ctrl, Alt, Shift)
- Special key methods (Enter, Tab, Backspace, Delete)
- Mouse button operations (left, right, middle)
- Relative and absolute mouse movement
- Find windows by title or process name
- Activate and manage windows
